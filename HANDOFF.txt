TECH STACK
  Plain static site. No framework, no build step, no package.json, no node_modules.
  HTML + CSS + vanilla JS. Tailwind via CDN (cdn.tailwindcss.com) with an inline
  tailwind.config in each page. Google Fonts: Lora (serif), Inter (sans),
  JetBrains Mono. Heading font stack prefers "Circular Std" then Helvetica/Arial
  (Circular Std is not bundled - falls back to Helvetica).
  All page data lives in people.js (a global PEOPLE object). No backend, no DB.

FOLDER STRUCTURE
  index.html      Home/feed: splash intro, color-cycle bg, white rising "sparks",
                  infinite category carousel (auto-scroll desktop / swipe on touch),
                  "Nominations Open" live counter, 21 vouch cards generated from people.js, footer.
  awards.html     "Categories" page (title/nav say Categories; file is still awards.html).
                  Big "*Bosses of Bangalore" heading + live "bosses nominated" counter w/ green LED,
                  category buttons (Overall/CXO/Engineering/Product), 4 poster cards each with a
                  top-5 live leaderboard preview + per-category "Nominate your boss for X" button.
  vouchwall.html  "Vouch Wall" - same vouch cards as the home feed.
  board.html      The category leaderboard (the actual "category pages"). Filter pills
                  (Overall/blue, Founders-CXOs/red, Eng/orange, Product/black) swap a body theme
                  class. White stars rise upward. Live overtake/+1 vouch animation. Reads ?cat=.
  profile.html    Individual boss profile, reads ?id=<slug>. Shows 20 homepage-style vouch

  nominate.html   Nomination form. Category select (CXO/Engineering/Product) pops a poster card.

RUN LOCALLY
  No package manager and nothing to install - it's static files.
  Serve the folder with any static server, e.g.:
    npx serve        (then open the printed URL, usually http://localhost:3000)
    or: python -m http.server 8000   (http://localhost:8000)
  Open index.html through the server (not file://) so relative paths and people.js load.
  No dev/build/watch step exists.
